# Cronômetro utilizando o arduino com um display de 4 dígitos e 7 segmentos 
Cronômetro bem simples onde e possível além de contar os segundos, marcar a hora atual do dia por meio dos botões do projeto, nesse código trabalhamos com uma variedade de conceitos que a programação em C proporciona como matrizes, variáveis globais, funções, passagem de parâmetro entre outros. Ficou curioso? quer ver a demostração desse projeto? então da uma olhada no nosso blog: [ardu.ino.sketch](https://instagram.com/ardu.ino.sketch?utm_source=ig_profile_share&igshid=q0plxzc5k6m4), la você irar encontra vários projetos interessantes com microcontroladores.
